PHP

PHP es un lenguaje de scripts del lado del servidor que permite a los desarrolladores crear contenido web dinámico e interactivo. A diferencia de HTML y CSS, que se utilizan para la estructura y el estilo, PHP es responsable de generar contenido dinámico y de manejar la lógica del lado del servidor, como interactuar con bases de datos y procesar datos de formularios.

Al usar PHP, los desarrolladores pueden incrustar scripts del lado del servidor dentro de HTML, lo que permite la generación dinámica de contenido basado en las interacciones del usuario y otras condiciones del servidor. Esto facilita la creación de aplicaciones web personalizables e interactivas que pueden responder a entradas del usuario y proporcionar experiencias personalizadas.

PHP también soporta la interacción con bases de datos a través de mecanismos como MySQL y PDO, lo que permite la creación de aplicaciones web basadas en datos que pueden recuperar, actualizar y almacenar información de manera eficiente. Esto conduce a soluciones web dinámicas y robustas. 

            
<?php
session_start();
include '../db.php';
if (!isset($_SESSION['loggedin'] ) || $_SESSION['loggedin'] !== true) {
header("Location: ../login/index.php");
exit();
}
$stmt = $conn->prepare("
SELECT p.id, p.post_content, p.post_image, p.created_at, u.username, u.profile_picture, 
       COUNT(c.id) AS comment_count
FROM posts p
JOIN usuarios u ON p.username = u.username
LEFT JOIN comments c ON p.id = c.post_id
GROUP BY p.id
ORDER BY p.id DESC
");
$stmt->execute();
$result = $stmt->get_result();
$posts = [];
while ($row = $result->fetch_assoc()) {
$posts[] = $row;
}
$stmt->close();
$conn->close();
?>
