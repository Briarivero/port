HTML

HTML, o Lenguaje de Marcado de Hipertexto, es el estándar fundamental para crear páginas web. Es el lenguaje que define la estructura y el contenido de una página web, utilizando etiquetas para identificar elementos como encabezados, imágenes, enlaces y más.

El propósito principal de HTML es estructurar la información en la web de manera coherente y accesible. Junto con CSS y JavaScript, forma la base del desarrollo web, permitiendo que los navegadores representen el contenido y permitiendo que los usuarios interactúen con él.

<html lang="en">
<head>
 <meta charset="UTF-8" />
 <link id="min-style" rel="stylesheet" type="text/css" href="css/styles.css" />
 <script src="js/archlinux.js"></script>
 <title>Brian Rivero</title>
</head>

 </body>
